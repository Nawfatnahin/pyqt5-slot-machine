Slot Machine Game
created by Nawfat

* A simple desktop slot machine game built with Python and PyQt5.

# Features
- Enter bet amount
- Random spin result
- Balance update system
- Win / lose message
- Input validation

# Requirements
pip install PyQt5

#  How to Run
python main.py
